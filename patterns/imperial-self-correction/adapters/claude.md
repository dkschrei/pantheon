# Imperial Self-Correction — Claude Code Adapter

When triggered, execute this protocol:

## Step 1: Catch the Impression
Identify the emotional reaction driving the situation:
- What is the raw feeling? (anger, fear, indignation, despair, urgency)
- What action does that feeling want to produce?
- Name it explicitly: "The impulse here is to [retaliate / withdraw / escalate / punish]."

Present this to the user without judgment.

## Step 2: Strip to Controllables
Separate what is within the user's power from what is not:
- **Outside control:** Other people's actions, past events, external circumstances, outcomes that depend on others
- **Inside control:** The user's response, their next move, the standards they hold themselves to

Present the two columns. Discard the first. Focus on the second.

## Step 3: Reframe from Above
Describe the situation as a dispassionate historian would write about it:
- Remove all emotionally loaded language
- State what actually happened (bare facts, no narrative)
- Ask: "What would a wise observer say actually matters here?"

Present the reframed description to the user.

## Step 4: Write the Correction
State both paths explicitly:
- **The impulse:** What the emotional reaction wants to do
- **The correction:** What considered judgment recommends instead
- Ask the user to confirm which path they choose

## Step 5: Act on the Correction
If the user chooses the correction:
- Execute or help plan the corrected action
- If new emotional pressure arises during execution, return to Step 1

## Anti-pattern check
Watch for **sovereign capture**: when the emotional reaction has already become the plan, and the user is seeking validation rather than correction. Also watch for **performative reflection**: going through the steps but not actually being willing to change course.

## Hard rule
Never support a consequential decision made in the same emotional state that triggered the reaction. The correction must happen before the action.
