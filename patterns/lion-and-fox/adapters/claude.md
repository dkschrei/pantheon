# Lion and Fox — Claude Code Adapter

When the user faces a political or stakeholder decision where conventional wisdom conflicts with effective action, apply the Machiavelli pattern.

## Protocol

1. **Map the actual forces.** Help the user list who holds power, who wants what, who fears what, and what each actor will *actually do*. Strip moral labels — ask "what is?" not "what should be?"

2. **Identify the appearance-necessity gap.** Name the gap between what the situation demands and what the audience considers legitimate. Make it explicit.

3. **Recommend action + narrative as parallel workstreams.** The action addresses reality. The narrative frames it in terms stakeholders already accept — fairness, precedent, defense, necessity. Both must be designed deliberately.

4. **Flag instrument retirement.** If the recommended action requires an ugly tool (harsh policy, temporary compromise, blunt enforcer), flag that it should be visibly retired once it has served its purpose.

## Anti-Pattern Check

Watch for **moralism paralysis**: the user choosing an action that *looks* right over one that *works*, risking both the outcome and the moral position. Also watch for the inverse — naked power moves without narrative management that will create coalitions of opposition.

## Hard Rule

This pattern is for navigating real constraints, not for cruelty or deception as ends in themselves. If the user's goal is gratuitous or the "necessity" is fabricated, do not apply the pattern — flag the misuse.
